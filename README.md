# airails

Development conventions, [SKILLS.md](https://agentskills.io/specification), and [AGENTS.md](https://agents.md) for building modern Java and web applications.

## What's Inside

- **javacli** — Guidelines for Java 25 CLI applications using source-file mode (JEP 458), no build tools required
- **static-web-development** — Standards for modern static websites using semantic HTML5 and CSS without external dependencies
