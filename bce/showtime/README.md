# showtime

An [AIrails.dev](https://airails.dev) skill for live coding sessions — works with any project skill (microprofile-server, java-cli-app, java-cli-script, etc.).

## Scope

- All coding and architecture rules from the active project skill apply
- Test execution (unit, integration, system) is skipped
- Build execution is skipped
- Focus on rapid code generation

## Usage

Activate during live demos, workshops, or rapid prototyping where verification overhead is unwanted. Combine with `/microprofile-server`, `/java-cli-app`, or any other project skill.
